TellMeMore (UBC HackDay 2016)

Winner of Best API Usage

TellMeMore is an iOS application that lets users generate tags of a given picture (using Clarifai's API). The user can then look at the list of tags generated for the image (sorted with by highest confidence) and click the tag to search the web to acquire more information about the tag. The idea was to create an application that tourists could use to easily find out more information about various attractions.

Built during the UBC 12-hour Local Hack Day, where it made the Top 3 projects as well as won the award for Best API Usage

Languages/Frameworks
Front-end: Objective-C, Swift  
Back-end: Python
Libraries: Clarifai

Buckets++ (QuackCon 2016)

Winner of Best Athletic Engagement

Buckets++ is a hybrid web application that uses a machine learning model (Gradient Boosting) to analyze CSVs of NBA players' shots and calculate the probability of the shot going in based on variables such as time remaining in game and opposition.

Languages/Frameworks
Front-end: HTML5, CSS3, JavaScript  
Back-end: Python, Flask, Node.js
Libraries: scikit learn, pandas, seaborn, matplotlib

LetsChill (HackArizona 2016)

LetsChill takes two locations, and identifies the middle point between the two. From there, it will search within a radius for places to meet such as restaurants, cafes, or even malls. It will list these options while displaying them on the map, and allow the user to filter the venue the user prefers. The list will adjust accordingly, thus providing the user with an easy interface to identify his/her next meetup location.

Languages/Frameworks
Front-end: HTML, CSS, JavaScript, jQuery, Google Material Design  
Back-end: Google AutoComplete API, Google Places API, Google Maps API

myCFL (SportsHack 2015)

During SportsHack, built a web application that allows fans to immerse themselves into a) the world of their favorite player if they are familiar with the CFL or b) have the choice to be matched with a player based of the commonality of personal information if they are new to the CFL. Used cosine-similarity (Machine Learning) to produce a list of top 5 most similar players to the fan based on given sign-up input data (alma matter, hometown, etc).

Languages/Frameworks
Front-end: HTML, CSS, JavaScript, jQuery  
Back-end: Parse API, Node.js, Python, Flask, Heroku (RESTful API)
Data Analysis: Cosine Similarity
Libraries: Various CFL .csv's and GeoPy

TutorMe (nwHacks 2016)

Students everywhere always get stumped, and need help in order to overcome struggles with their studies.
